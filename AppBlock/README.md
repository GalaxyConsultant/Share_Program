📘 StudyGuard (공신폰 모드)

StudyGuard는 학습 집중 환경을 만들기 위한 안드로이드 전용 공신폰 앱입니다.
접근성 서비스와 기기 관리자(Device Admin)를 활용하여
설정, 앱 설치, 스토어 접근 등을 제한함으로써
아이 또는 학습자가 임의로 환경을 변경하지 못하도록 설계되었습니다.

⚠️ 본 앱은 플레이스토어 배포용 앱이 아닙니다.
APK 직접 설치 방식으로만 사용을 권장합니다.

🔐 주요 기능
1. 공신폰 모드 활성화

접근성 서비스 활성화 후 공신폰 모드를 실행하면

아래 항목들이 즉시 제한됩니다:

⚙️ 시스템 설정 앱

📁 파일 관리자 (APK 설치 우회 방지)

🛒 Play 스토어

📦 패키지 설치 관련 앱

접근성 이벤트를 감지하여 차단 대상 앱 진입 시 자동으로 뒤로가기 처리합니다.

2. 관리자 해제 (비밀번호 필요)

공신폰 모드 해제는 관리자 화면에서만 가능

서버에서 내려받은 관리자 비밀번호 검증 후 해제

비밀번호는 숫자/문자 혼합 입력 가능

3. 기기 관리자(Device Admin) 등록

앱 삭제 방지를 위해 기기 관리자 권한을 등록합니다.

일반적인 방식으로는 앱 삭제가 불가능합니다.

🔒 기기 관리자 권한 해제 후에만 앱 삭제 가능

⚠️ 알려진 제한 사항 (중요)
❗ 연타 시 앱 삭제 가능성

일부 기기/OS 버전에서
설정 → 앱 정보 진입 후 매우 빠른 연타 조작을 할 경우
앱 삭제 화면에 접근 가능한 사례가 확인되었습니다.

이는:

안드로이드 시스템 동작 특성

접근성 서비스의 구조적 한계

로 인해 완전 차단은 불가능합니다.

❗ 이는 앱의 결함이 아니라
❗ 안드로이드 OS 보안 정책상 허용된 동작입니다.

📵 플레이스토어 배포 관련

본 앱은 다음 사유로 플레이스토어 등록이 불가능합니다:

접근성 서비스를 이용한 시스템 앱 차단

사용자 설정/삭제 권한 제한

보호자/관리자 통제 목적의 기기 제어

👉 외부 배포 전용 앱으로 사용해 주세요.

🛠 권장 사용 환경

Android 9 이상

삼성 갤럭시 계열 권장 (One UI)

공기계 또는 학습 전용 기기 사용 권장

⚖️ 법적/책임 고지

본 앱은 학습 보조 목적으로 제작되었습니다.

불법 감시, 도청, 무단 통제 목적 사용을 금합니다.

사용자의 기기 설정 및 관리 책임은 사용자 본인에게 있습니다.
