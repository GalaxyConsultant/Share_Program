# 📱 Mobile ADB

폰 ↔ 폰 무선 ADB 연결을 간편하게 도와주는 모바일 전용 유틸리티

---

## 🚀 소개

Mobile ADB는 PC 없이 스마트폰끼리 무선 디버깅(ADB Wi-Fi)을 연결할 수 있도록  
명령어 생성과 이력 관리를 도와주는 앱입니다.

Termux 환경에서 바로 사용할 수 있도록  
필요한 명령어를 자동으로 준비해줍니다.

---

## ✨ 주요 기능

- 📡 무선 ADB 연결 명령어 생성
- 🔑 페어링 코드 기반 연결 지원
- 📋 명령어 원터치 복사
- 🗂 연결 이력 자동 저장
- 🗑 이력 삭제 기능
- 📦 Termux 설치 링크 제공
- ⚙ 초기 설정 스크립트 자동 생성

---

## 🛠 사용 방법

### 1️⃣ Termux 설치
앱 내 버튼을 통해 최신 Termux를 설치합니다.

### 2️⃣ 초기 설정
초기화 명령어를 복사 후 Termux에 붙여넣어 실행합니다.

### 3️⃣ 무선 디버깅 활성화
설정 → 개발자 옵션 → 무선 디버깅 활성화

### 4️⃣ IP · 포트 입력
무선 디버깅 화면에 표시되는 IP 및 포트를 입력합니다.

### 5️⃣ 페어링 코드 입력
표시된 6자리 페어링 코드를 입력합니다.

### 6️⃣ 명령어 복사 후 실행
생성된 명령어를 Termux에 붙여넣으면 연결 완료됩니다.

## 📂 연결 이력 관리

- 최근 연결 기록 자동 저장
- 탭 → 입력창 자동 채움
- 길게 누르기 → 기록 삭제
