🔒 소개 (Introduction)



Not Forensics Suite 는

일반 사용자가 초기화 없이 기기(안드로이드 스마트폰 / PC)를 양도하거나 판매할 때,

이미 삭제된 데이터의 잔여 흔적이 포렌식 도구에서 복구되지 않도록

안전하게 처리해주는 보안 도구입니다.



스마트폰 버전: Not Forensics Android



PC 버전: Not Forensics PC



두 프로그램 모두 동일한 보안 목표를 가지고 있으며,

사용자가 직접 삭제한 파일 및 삭제 흔적이 타인에 의해 복구되는 위험을 최소화하도록 설계되었습니다.
