# 고객관리 프로그램

오프라인 매장에서 고객 상담 내용을 간단하게 기록하고,  
엑셀 파일로 정리 · 관리할 수 있도록 만든 데스크톱용 고객관리 프로그램입니다.  

윈도우에서 실행되는 단일 EXE / 파이썬 프로그램 형태로,  
**고객 정보 입력 → 목록 확인 → 검색/수정 → 엑셀 저장/출력** 흐름을 지원합니다.

---

## 주요 기능

### 1. 인증키 기반 실행

- 프로그램 실행 시 **인증키 입력 창**이 먼저 표시됩니다.
- 유효한 인증키를 한 번 입력하면,  
  그 시점 기준으로 **30일 동안 자동으로 인증 상태를 유지**합니다.
- 30일이 지나면 인증 정보가 만료되고, 다음 실행 시 다시 인증키를 입력해야 합니다.
- 인증키가 없거나 분실한 경우, **“인증키 문의” 버튼**을 눌러 안내된 연락처로 문자 요청할 수 있습니다.

> 👉 이 과정을 통해 무단 사용을 막고, 정식 사용자만 프로그램을 사용할 수 있도록 합니다.

---

### 2. 고객 정보 입력 폼

상단에는 고객 한 명의 상담/방문 정보를 입력하는 폼이 있습니다.

입력 항목:

- **성함**
- **연락처**
- **내용 (구매 및 정보)**
- **특이사항**
- **비고**

특징:

- 성함/연락처는 한 줄 입력(Entry) 방식
- 내용/특이사항/비고는 여러 줄 입력(Text) 방식
- `Tab` 키로 입력 칸 사이를 자연스럽게 이동할 수 있어,  
  마우스 없이 빠르게 입력할 수 있습니다.
- “입력창 지우기” 버튼으로 모든 입력 칸을 한 번에 초기화할 수 있습니다.

---

### 3. 고객 목록 테이블 (Treeview)

하단에는 **고객 데이터 목록**이 표 형태로 표시됩니다.

컬럼:

- 날짜
- 시간
- 성함
- 연락처
- 내용 (구매 및 정보)
- 특이사항
- 비고

특징:

- 상단 폼에서 “추가” 버튼을 누르면, 현재 시간(날짜/시간) 기준으로 한 줄이 목록에 추가됩니다.
- 목록에서 특정 행을 클릭하면,  
  해당 행의 데이터가 다시 위쪽 입력 폼에 자동으로 채워집니다.
- “수정” 버튼을 누르면, 선택된 행의 내용이 입력 폼 기준으로 덮어써집니다.
- “삭제” 버튼을 누르면, 선택된 행이 목록에서 제거됩니다.
- 목록에 있는 데이터는 메모리에서 관리되며,  
  “저장” 버튼을 눌러야 엑셀 파일에 실제로 반영됩니다.

---

### 4. 검색 기능

버튼 영역 왼쪽에는 **검색창**이 있습니다.

- 검색어를 입력한 뒤 `Enter` 키를 누르면,  
  목록에 있는 전체 데이터 중 **해당 검색어를 포함하는 행만** 필터링해서 보여줍니다.
- 검색 대상은
  - 날짜
  - 시간
  - 성함
  - 연락처
  - 내용 / 특이사항 / 비고  
  등 **모든 컬럼 텍스트 전체**입니다.
- 검색창을 비우고 다시 `Enter`를 누르면,  
  전체 목록이 다시 표시됩니다.

> 예시  
> - 성함으로 검색: “홍길동”  
> - 연락처 일부로 검색: “010-1234”  
> - 특정 키워드로 검색: “환불 문의”, “A/S 요청” 등

---

### 5. 엑셀 파일 연동

프로그램의 핵심 기능 중 하나는 **엑셀 파일과의 연동**입니다.

#### 5-1. 엑셀 불러오기 (새로 생성)

- “엑셀 불러오기(새로 생성)” 버튼을 누르면  
  기존에 사용할 엑셀 파일(.xlsx)을 선택할 수 있습니다.
- 선택한 파일은 다음과 같이 동작합니다:
  - 파일이 없으면 새로 생성
  - 파일은 존재하지만 시트 구조가 맞지 않으면,  
    프로그램에서 사용하는 **전용 시트** 구조로 다시 만들어 줍니다.
- 이 과정에서 **기존 데이터가 초기화**되므로,  
  진행 전 경고 메시지로 한 번 더 확인을 요청합니다.

#### 5-2. 엑셀 구조 & 저장 방식

- 엑셀 파일에는 고정된 시트 이름(예: `고객데이터`)이 사용됩니다.
- 첫 줄에는 프로그램에서 사용하는 **헤더(열 제목)** 이 자동으로 세팅됩니다.
- “저장” 버튼을 누르면:
  - 기존 데이터 영역(2행 이후)을 모두 지우고
  - 현재 프로그램에 올라와 있는 모든 고객 데이터를  
    순서대로 다시 기록합니다.
- 내용/특이사항/비고는 엑셀에서 **줄바꿈(wrap_text)** 이 적용되도록 설정되어,  
  인쇄 시에도 내용이 잘려 보이지 않게 구성되어 있습니다.

#### 5-3. A4 출력에 맞춰진 서식

엑셀 새로 생성 시, 기본적으로 다음과 같이 설정됩니다.

- 용지 크기: **A4**
- 방향: 세로 (portrait)
- 너비 기준 맞춤: 가로 폭에 맞게 자동 축소/확대(fit to width)
- 적당한 여백 및 컬럼 너비가 설정되어,  
  인쇄 시 바로 사용하기 적합한 형태로 생성됩니다.

---

### 6. 설정 파일 (마지막 엑셀 파일 기억)

프로그램은 사용 편의를 위해 간단한 설정을 함께 저장합니다.

- 마지막으로 사용한 **엑셀 파일 경로**를 별도 설정 파일에 기록합니다.
- 다음에 프로그램을 실행하면,  
  사용자가 다시 파일을 지정하지 않아도 **이전에 쓰던 엑셀 파일을 자동으로 불러옵니다.**
- 만약 해당 파일이 삭제되었거나 이동된 경우에는,  
  엑셀 파일 선택 과정을 다시 진행하면 됩니다.

---

## 사용 흐름 요약

1. **프로그램 실행**
   - 인증키 입력 창 표시
2. **인증키 입력**
   - 유효하면 30일 동안 재입력 없이 사용 가능
3. **엑셀 파일 지정**
   - 처음 실행 시 또는 새 파일이 필요할 때  
     “엑셀 불러오기(새로 생성)” 버튼으로 파일 선택/생성
4. **고객 정보 입력 & 추가**
   - 폼에 정보 입력 후 “추가” 버튼으로 목록에 추가
5. **검색 / 수정 / 삭제**
   - 검색창으로 원하는 고객/내용 필터링
   - 목록 선택 후 “수정” / “삭제” 버튼 사용
6. **저장**
   - “저장” 버튼으로 현재 목록을 엑셀에 반영
7. **다음 실행**
   - 인증 유효기간 내라면 바로 실행
   - 마지막에 사용한 엑셀 파일을 자동으로 불러와 이어서 작업 가능

## 주의사항

- “엑셀 불러오기(새로 생성)” 기능은  
  선택된 엑셀 파일의 고객 데이터 영역을 **초기화**하고 새로 구성합니다.  
  → 기존 데이터를 유지하려면, **사용 전 백업**을 권장합니다.
- 인증키는 PC 기준으로 30일 동안만 유효합니다.  
  기간이 지나면 다시 인증이 필요합니다.
- 엑셀 파일이 네트워크 드라이브나 외부 장치에 있을 경우,  
  연결이 끊기면 저장 시 오류가 발생할 수 있으니 주의가 필요합니다.
